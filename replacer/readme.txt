* Description
Background replacement for a photo that taken with a green screen.

* Environment
* Describe the OS and compilation requirements needed to compile and run the program
* How to Run Your Program
1) Before running the code:
    • Make sure you have the necessary images in picture folder (img).
    • Make sure you have python version 3.10.8
    • Install openCV using pip install opencv-contrib-python.
    • Install numpy using pip install numpy.

2) To run this program, enter the following into the command prompt:

    python replacer.py .\images\<picture with a green screen name>.jpg .\images\<background picture name>.jpg <out put name picture>.jpg
