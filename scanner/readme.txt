* Description
A program that accepts a photo of a rectangular page (like a 4A page or a photo of a book)
and straightens the photo - similar to how the CamScanner application works.

* Environment
* Describe the OS and compilation requirements needed to compile and run the program
* How to Run Your Program
1) Before running the code:
    • Make sure you have the necessary images in picture folder (input).
    • Make sure you have python version 3.10.8
    • Install openCV using pip install opencv-contrib-python.
    • Install numpy using pip install numpy.
    • Make sure that you are in the folder where the Scanner.py is

2) To run this program, enter the following into the command prompt:

    python.exe Scanner.py .\input\<input_img> <path_output_img>
